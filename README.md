Project Title

Python Fundamentals: Lists, Dictionaries, Sets, Operators & Conditional Statements

Simple Overview

This project demonstrates basic Python programming concepts, including list creation, modification, and access, dictionary operations, set operations, and conditional statements.
It is designed for beginners learning Python fundamentals.

Description

This project contains Python code snippets that help learners understand how to work with essential data structures such as lists, dictionaries, and sets, along with writing simple conditional programs.
Each section includes examples showing how these data structures can be created, modified, and accessed.
The final part contains an interactive performance category program that uses conditional statements and user input.

Getting Started
Dependencies

Before running the program, ensure that you have:

Python 3.8 or above

Windows 10 / Linux / macOS

A code editor (VS Code, PyCharm, IDLE, etc.)

No external libraries are required.

Installing

To run this project:

Download or copy the Python script file.

Save the file as:

assignment2.ipynb


No additional installation steps are required.

Executing Program
How to Run the Program

Open a terminal or command prompt.

Navigate to the folder containing assignment2.ipynb.

Run the command:

python assignment2.ipynb
